### Run your first scan

AI-SLOP Detector measures structural risk in AI-generated code — unimplemented
stubs, phantom imports, disconnected pipelines, clone clusters, and inflated
docs. No LLM in the scoring path; the result is deterministic.

Open the Command Palette and run **SLOP Detector: Analyze Workspace**, or click
the search icon in the SLOP sidebar.

| What you get | Where |
|----------|---------|
| Per-file deficit score (0–100) and status band | Sidebar + status bar |
| 4D measurement (LDR · Inflation · DDC · Purity) | File tree metric row |
| Pattern findings with line references | Problems panel + tree |
